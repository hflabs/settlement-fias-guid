Данные предоставлены HFLabs.

Данные содержат 2 набора:
* RussianOikonymsDatasetFiasGuid.tsv — датасет Яндекса RussianOikonymsDataset, обогащенный уникальными GUID из ФИАС;
* RussianOikonymsDatasetUnofficialFiasGuid.tsv — датасет Яндекса RussianOikonymsDataset, обогащенный уникальными GUID из ФИАС.

Список RussianOikonymsDatasetFiasGuid.tsv содержит 160183 строки.
Список RussianOikonymsDatasetUnofficialFiasGuid.tsv содержит 60651 строку.
Кодировка файлов UTF-8.

Столбцы датасета:
* name — название населённого пункта. В большинстве случаев начинается с типа населённого пункта (деревня, село и т.д).
* other_name — другие названия, прежние названия, названия на других языках. Заполнено не для всех населённых пунктов.
* region_name — субъект федерации, в котором находится населённый пункт.
* district — муниципальное образование, в котором находится населённый пункт.
* lon — долгота.
* lat — широта.
* population — население.
* approx — признак того, что население указано приблизительно. 0 — точное население, 1 — приблизительное.
* fias_guid — ФИАС ГУИД адреса.


Данные наборы не предназначены для коммерческого использования. 

Оригинальный источник населенных пунктов, не обогащенный уникальными GUID из ФИАС, доступен по ссылке: https://yastatic.net/s3/milab/2021/toponyms/data/RussianOikonymsDataset.zip.
